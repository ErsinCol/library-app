# Library App

This project was developed as part of The Odin Project's Node.js and JavaScript Library lesson. This application is a simple JavaScript program that allows users to store books as a library.

Project link: https://www.theodinproject.com/lessons/node-path-javascript-library

### Technologies Used
- HTML5
- CSS3
- JavaScript, Javascript Classes
